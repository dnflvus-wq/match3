# Phase 2: 특수 타일 시스템

## 목표
4매치/5매치/L자/T자 매치 시 특수 타일 생성 + 조합 시너지 구현

## 특수 타일 종류

| 매치 형태 | 생성 타일 | 효과 |
|----------|---------|------|
| 4매치 직선 | 줄무늬 (Striped) | 가로 or 세로 한 줄 전체 파괴 |
| L자/T자 (5개) | 폭탄 (Wrapped) | 3x3 범위 2회 폭발 |
| 5매치 직선 | 레인보우 (Color Bomb) | 교환한 색상 전체 파괴 |
| 2x2 정사각형 | 로켓 (Rocket) | 장애물 추적 타격 |

## 조합 시너지

| 조합 | 효과 |
|-----|------|
| 줄무늬 + 줄무늬 | 십자(+) 폭발 |
| 줄무늬 + 폭탄 | 3줄 × 십자 광역 |
| 폭탄 + 폭탄 | 5x5 2회 폭발 |
| 레인보우 + 특수 | 해당 색상 전부 특수 타일로 변환 후 동시 발동 |
| 레인보우 + 레인보우 | 보드 전체 싹쓸이 |

## 구현 방식
- **ScriptableObject**: 각 특수 타일의 스프라이트, 이펙트 Prefab, 폭발 반경, 점수
- **Strategy 패턴**: ISpecialTileEffect 인터페이스, 각 타일별 구현 클래스
- **MatchFinder 확장**: 매치 형태(Shape) 인식 로직 추가
- **Prefab**: 특수 타일별 오버레이 스프라이트 + 전용 파티클

## 보너스 타임 (레벨 클리어 시)
- 남은 이동수 → 무작위 특수 타일로 변환 → 연쇄 폭발
- 캔디크러쉬의 "Sugar Crush"와 동일한 연출
- 추가 점수 + 시각적 쾌감 극대화

## 검증 항목
- [ ] 4매치 → 줄무늬 타일 생성

- [ ] L/T자 → 폭탄 생성
