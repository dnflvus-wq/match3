# Phase 3: 튜토리얼 시스템

## 목표
신규 플레이어를 위한 단계별 인터랙티브 튜토리얼

## 단계별 설계

### Step 1: 기본 스와이프
- 고정 배열 보드, 매치 가능한 곳 1개만 존재
- 반투명 검은 마스크로 나머지 어둡게
- 손가락 아이콘이 스와이프 방향 안내 (DOTween Ease.OutQuad)
- 매치 성공 → 마스크 해제 + 보상 연출

### Step 2: 특수 타일
- 4매치 가능한 배열 제공
- "4개를 맞추면 특수 타일이 만들어져요!" 힌트
- 특수 타일 터뜨려보기

### Step 3: 부스터
- 교착 상태 인위적 생성
- 부스터 UI 강조 → 사용 유도
- "턴 소모 없이 사용할 수 있어요!"

## 구현 방식
- **TutorialStepSO (ScriptableObject)**: 대상 좌표, 스와이프 방향, 허용 입력, 힌트 텍스트
- **마스킹**: Canvas 최상단 반투명 패널 + Sprite Mask로 구멍
- **손가락 애니메이션**: UI Image + DOTween 반복 이동
- **FSM 연동**: 튜토리얼 중에는 허용된 입력만 통과
