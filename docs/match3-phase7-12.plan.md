캔디크러쉬 수준의 상용화 완성

## Phase 7: 오브젝트 풀링 (GC 방지)

### 7-1. 타일 풀링
- Instantiate/Destroy → 풀에서 꺼내/반환으로 전환
- 색상별 풀 (각 20개 사전 할당)
- GameGrid.SpawnNewPiece → Pool.Get()
- ClearablePiece.Destroy → Pool.Return()

### 7-2. 파티클 풀링
- 파티클용 별도 풀 (30개)
- 재사용 후 자동 비활성화

### 7-3. 검증 항목
- [ ] 프로파일러에서 GC 스파이크 없음
- [ ] 100회 매치 후 메모리 안정
- [ ] 풀 부족 시 자동 확장
- **커밋:** `perf: 오브젝트 풀링 도입`

## Phase 8: 파티클 이펙트

### 8-1. 매치 스파크
- 타일 제거 위치에서 별 모양 파티클 방출
- 색상별 파티클 색상 매칭
- Burst 모드 (한 번에 5~8개)

### 8-2. 특수 타일 폭발
- 줄무늬: 가로/세로 라인 이펙트
- 폭탄: 원형 폭발
- 레인보우: 무지개 빛 산개

### 8-3. 콤보 이펙트
- 연쇄 2회: 작은 스파크
- 연쇄 3회: 중간 폭발
- 연쇄 5회+: 화면 전체 이펙트

### 8-4. 검증 항목
- [ ] 파티클이 타일 색상과 일치
- [ ] 60fps 유지 (파티클 과다 방지)
- [ ] 풀링으로 GC 없음
- **커밋:** `feat: 파티클 이펙트 시스템`

## Phase 9: 오디오 시스템

### 9-1. AudioManager 싱글톤
```csharp
public class AudioManager : MonoBehaviour {
    public static AudioManager Instance;
    public AudioClip matchSound, swapSound, failSound, comboSound, bgm;
}
```

### 9-2. 효과음 목록
| 이벤트 | 소리 | 비고 |
|---|---|---|
| 매치 성공 | 짤깍/팡 | 기본 |
| 스왑 | 슉 | 짧게 |
| 실패 (핑퐁) | 뿅 | 거부감 |
| 콤보 | 피치 상승 | pitch = 1.0 + combo * 0.1 |
| 특수 타일 생성 | 차밍 | 보상감 |
| 레벨 클리어 | 팡파레 | 화려하게 |
| 게임 오버 | 슬픈 | 짧게 |

### 9-3. BGM
- 메인 메뉴: 밝고 경쾌한 루프
- 인게임: 긴장감 있는 루프
- 이동 부족 시: 템포 상승

### 9-4. 검증 항목
- [ ] 모든 이벤트에 소리 재생
- [ ] 콤보 시 피치 점진 상승
- [ ] BGM + 효과음 동시 재생
- [ ] 음소거 토글 동작
- **커밋:** `feat: 오디오 시스템 (효과음 + BGM)`

## Phase 10: 특수 타일 강화

### 10-1. 줄무늬 캔디 시각
- 가로 줄무늬: 기본 캔디 + 가로선 오버레이
- 세로 줄무늬: 기본 캔디 + 세로선 오버레이
- 생성 시 반짝이 연출

### 10-2. 폭탄 (T/L 매치)
- T자 또는 L자 매치 → 폭탄 생성
- 3x3 범위 폭발
- 폭탄 이미지: 해당 색상 + 폭발 이펙트

### 10-3. 특수 + 특수 조합
| 조합 | 효과 |
|---|---|
| 줄무늬 + 줄무늬 | 십자 폭발 |
| 줄무늬 + 폭탄 | 3줄 가로+세로 |
| 폭탄 + 폭탄 | 5x5 범위 |
| 레인보우 + 일반 | 해당 색 전부 제거 (기존) |
| 레인보우 + 줄무늬 | 전부 줄무늬로 변환 후 폭발 |
| 레인보우 + 폭탄 | 전부 폭탄으로 변환 후 폭발 |
| 레인보우 + 레인보우 | 전체 보드 클리어 |

### 10-4. 검증 항목
- [ ] T/L 매치 시 폭탄 생성
- [ ] 특수 타일 조합 효과 정상
- [ ] 이미지 정상 표시
- **커밋:** `feat: 특수 타일 강화 (폭탄, 조합)`

## Phase 11: 레벨 시스템

### 11-1. 레벨 에디터/데이터
- ScriptableObject (LevelData.cs 이미 생성됨) 활용
- 10개 레벨 추가 (난이도 점진 상승)
- 장애물 배치 (빙하, 잠금 타일, 체인 등)

### 11-2. 레벨 선택 화면
- 맵 스크롤 (캔디크러쉬 스타일 경로)
- 별점 표시 (1~3성)
- 잠금/해금 상태

### 11-3. 난이도 곡선
| 레벨 | 이동수 | 색상수 | 특징 |
|---|---|---|---|
| 1~3 | 25~30 | 4 | 튜토리얼, 쉬움 |
| 4~6 | 20~25 | 5 | 장애물 등장 |
| 7~9 | 18~22 | 5 | 복합 장애물 |
| 10 | 15 | 6 | 보스 레벨 |

### 11-4. 튜토리얼
- 레벨 1: 스와이프 안내 (손가락 가이드)
- 레벨 2: 특수 타일 설명
- 레벨 3: 콤보 설명

### 11-5. 검증 항목
- [ ] 10개 레벨 플레이 가능
- [ ] 별점 저장/표시
- [ ] 난이도 체감
- **커밋:** `feat: 10레벨 + 튜토리얼 + 맵 화면`

## Phase 12: 메타 시스템 (상용화)

### 12-1. 라이프/하트 시스템
- 하트 5개, 실패 시 -1
- 30분마다 자동 충전

### 12-2. 부스터
| 부스터 | 효과 | 획득 |

| 무지개 폭탄 | 한 색상 전체 제거 | 광고 시청 |
| +5 이동 | 이동 5회 추가 | 코인 구매 |
| 셔플 | 보드 셔플 | 코인 구매 |

### 12-3. 코인 시스템

- 부스터 구매
- 일일 보상

### 12-4. 설정 화면
- BGM 온/오프
- 효과음 온/오프
- 진동 온/오프
- 크레딧

### 12-5. 검증 항목
