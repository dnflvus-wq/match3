# Phase 8: GDD (Game Design Document)

> 이 문서는 Match3 게임의 **전체 규칙, 숫자, 콘텐츠**를 정의한다.
> 모든 수치는 Candy Crush, Royal Match 등 상용 게임 기준 (NotebookLM 리서치 기반).

## 1. 게임 개요
- **장르**: Match3 퍼즐
- **플랫폼**: Android (향후 iOS)
- **타겟 유저**: 캐주얼 모바일 게이머 (25~45세, 여성 비율 70%)
- **핵심 재미**: 연쇄 반응의 시각적 쾌감, 전략적 특수 타일 조합
- **테마**: 판타지 (마법의 숲)
- **세션 길이**: 레벨 당 평균 2~3분

## 2. 코어 게임플레이 규칙

### 스와이프
- 인접한 **상하좌우 4방향**으로만 1칸 스와이프 (대각선 금지)
- 매치 불성립 시 **0.25초** 핑퐁 애니메이션으로 원위치
- 드래그 임계값: 타일 크기의 **50%** 이상 이동해야 인식

### 매치
- **최소 3개** 동일 색상 타일이 직선(가로/세로)으로 정렬
- 매치된 타일은 즉시 파괴 → 빈칸 발생

### 연쇄 반응 (Cascade)
- 타일 파괴 → 상단 타일 **중력 낙하** (속도: 8 units/sec, Ease: OutBounce)
- 새 타일은 보드 위 **2유닛** 높이에서 스폰 → 낙하
- 낙하 후 새 매치 발생 시 **자동 연쇄** (무한 반복)
- 연쇄 중 **입력 차단** (FSM: EVALUATE/MATCH/COLLAPSE 상태)

### 턴 종료 조건
- 모든 매치 + 낙하 + 연쇄가 완전히 종료된 **안정화(Stable) 상태**에서만 1턴 소비
- 입력이 유효한 매치를 만들지 못한 경우 → 턴 소비 없음 (핑퐁만)

### 힌트
- **10초** 동안 무조작 → 유효 스왑 타일 2개가 **0.5초 주기**로 펄스 애니메이션
- Scale: 1.0 ↔ 1.15, Ease: InOutSine

### 데드 보드
- 보드에 유효한 이동이 **0개** → 자동 셔플
- 셔플 후에도 유효 이동 없으면 → 재셔플 (최대 3회)

## 3. 특수 타일

### 생성 조건

| 매치 형태 | 생성 타일 | 효과 범위 | 점수 배수 |
|-----------|----------|----------|----------|
| **4매치 직선** (가로/세로) | 스트라이프 (Striped) | 스와이프 방향 기준 1줄 전체 (1×9 또는 9×1) | ×2 |
| **5매치 L자/T자 교차** | 폭탄 (Wrapped/Bomb) | 중심 기준 **3×3** (9칸), 2회 연속 폭발 | ×3 |
| **5매치 직선** | 컬러밤 (Color Bomb) | 스왑 대상과 같은 색 **전체 제거** | ×4 |
| **2×2 정사각형** | 유도형 (Paper Plane) | 십자 4칸 + 목표물 1개 추적 타격 | ×2 |

### 조합 시너지 (두 특수 타일 스왑)

| 조합 | 효과 |
|------|------|
| 스트라이프 + 스트라이프 | 스왑 위치 중심 가로+세로 **거대 십자(+)** |
| 스트라이프 + 폭탄 | **3줄 두께** 거대 십자가 (가로3+세로3) |
| 폭탄 + 폭탄 | **5×5** 범위 (25칸), 2회 연속 폭발 |
| 컬러밤 + 일반 타일 | 해당 색상 **전체** 보드에서 제거 |
| 컬러밤 + 스트라이프/폭탄/유도형 | 해당 색 전체 → 스와이프한 특수 타일로 변환 → 동시 기폭 |
| 컬러밤 + 컬러밤 | **보드 전체** 1회 싹쓸이 + 모든 장애물 1단계 파괴 |

## 4. 승리/패배 조건

| 모드 | 승리 | 패배 | 비고 |
|------|------|------|------|
| **이동수(Move)** | 제한 이동수 내 미션 100% 달성 | 이동수 0 도달 시 미달성 | 미션 예: 사과 15개, 바나나 20개 수집 |
| **시간(Time)** | 제한 시간 내 미션 달성 | 타이머 0초 도달 | 연쇄 시 +1초, 시간 보너스 타일 +5초 |
| **장애물(Obstacle)** | 제한 이동수 내 장애물 전부 파괴 | 이동수 0 도달 | 얼음 3겹, 사슬, 금속 등 |

### 보너스 타임 (레벨 클리어 후)
- 남은 이동수 1회당 → 보드 위 일반 타일 1개가 **무작위 특수 타일로 변환 → 기폭**
- 변환 1회당 **1,000점** + 콤보 연쇄 폭발 점수

## 5. 점수 시스템

| 항목 | 점수 |
|------|------|
| 일반 타일 1개 파괴 | **20점** |
| 3매치 기본 | **60점** (20 × 3) |
| 4매치 | **80점** (20 × 4) |
| 5매치 | **100점** (20 × 5) |
| 연쇄 보너스 | 1연쇄 ×1, 2연쇄 ×2, 3연쇄 ×3 ... (누적) |
| 스트라이프 기폭 | **+120점** 고정 + 파괴 타일당 점수 |
| 폭탄 기폭 | **+200점** 고정 + 파괴 타일당 점수 |
| 컬러밤 기폭 | **+300점** 고정 + 파괴 타일당 ×4 |
| 남은 이동수 보너스 | 1회당 **1,000점** + 특수 타일 연쇄 |

### 별 등급 (레벨 클리어 시)
- ★☆☆: 목표 달성 (targetScore 도달)
- ★★☆: targetScore × **1.5배**
- ★★★: targetScore × **2.5배**

## 6. 부스터

| 부스터 | 효과 | 가격 (코인) | 획득 방법 |
|--------|------|------------|----------|
| **망치** (Hammer) | 지정 타일 1개 또는 장애물 1단계 파괴 | 900 | 일일 룰렛, 출석 보상 |
| **셔플** (Shuffle) | 보드 전체 일반 타일 랜덤 재배치 (장애물 제외) | 600 | 레벨 구간 돌파 보상 |
| **+5 이동수** | 패배 직전 팝업 → 5회 추가 턴 | 900 (연속 구매 시 +1000씩 인플레이션) | 보상형 광고 1회 시청으로 대체 가능 |

### 코인 경제
- 레벨 클리어 보상: **50~150 코인** (별 등급에 비례)
- 일일 보상: **100~500 코인** (연속 출석 보너스)
- 코인 IAP: $0.99 = 500코인, $4.99 = 3000코인, $9.99 = 8000코인

## 7. 레벨 구조

### 목표
- 런칭 시 **500개 이상** → 주간 40~80개 추가 → 최종 10,000+개
- 월드 1개 = **15~20레벨**, 월드 전환 시 신규 기믹 1종 해금

### 난이도 곡선 (승률 기준)

| 구간 | 레벨 | 승률 | 특징 |
|------|------|------|------|
| Easy & Fun | 1~20 | **95%+** | 튜토리얼, 결제 유도 없음, 콤보 쾌감 위주 |
| Mildly Challenging | 21~50 | **70%** | 사슬/1단 얼음 등장, 약한 전략 필요 |
| Engaging | 51~100 | **50%** | 다중 레이어 장애물, 재도전 요구 |
| Tactical Mastery | 101~200 | **30~40%** | 특수 타일 조합 세팅 필수 |
| Hardcore (Fuu-effect) | 200+ | **10~20%** | 극한 난이도, +5이동수/부스터 IAP 유도 구간 |

## 8. 장애물 종류

| 장애물 | HP | 파괴 방법 | 등장 시점 |
|--------|----|---------|---------|
| **얼음** (Ice) | 1~3겹 | 인접 매치 1회당 1겹 파괴 | Level 21+ |
| **사슬** (Chain) | 1~2겹 | 인접 매치로 1겹씩 해제 | Level 31+ |
| **금속** (Stone) | 파괴 불가 | 이동 불가, 빈칸처럼 우회 | Level 41+ |
| **젤리** (Jelly) | 1~2겹 | 위에서 매치 발생 시 1겹 제거 | Level 51+ |
| **포털** (Portal) | - | 타일이 통과하여 다른 위치로 이동 | Level 61+ |
| **타이머 폭탄** | N턴 | N턴 내 인접 매치로 제거, 미제거 시 패배 | Level 71+ |

## 9. UI/UX 화면 흐름

```
앱 실행 → 로딩 → 레벨 선택 → 레벨 시작 팝업(목표+부스터 선택)
→ 인게임(HUD+보드+부스터바) → 클리어 시: 보너스 타임→결과(별+점수)
→ 실패 시: +5이동수 팝업 → 재도전/홈
```

### 화면별 핵심 요소
- **레벨 선택**: 월드맵 스크롤, 레벨 버튼(별 표시), 잠금 상태
- **레벨 시작 팝업**: 목표 아이콘(사과/얼음 등), 부스터 3종 ON/OFF 선택
- **인게임 HUD**: 점수(좌), 남은이동수(중), 목표(우)
- **인게임 하단**: 부스터 3종 + 코인 잔액
- **결과 팝업**: 별 1~3 순차 애니메이션, 점수 카운트업, Replay/Home 버튼

## 10. 아트 스타일
- **테마**: 마법의 숲 (판타지)
- **타일**: 물고기 6색 (fish_1~6), 향후 8색 확장
- **배경**: 레벨별 5종 테마 (숲/동굴/호수/성/하늘)
- **UI**: 나무판 스타일 (hud_panel.png), 따뜻한 황금색 톤
- **파티클**: 스파클/별/글로우/링 4종

## 11. 사운드
- **BGM**: 판타지 오케스트라 + 오르골 (레벨 선택: 밝음, 인게임: 긴장감)
- **SFX**: 매치(뽕), 스왑(슈웅), 콤보(차임 상승음), 특수타일(강렬한 폭발), 게임오버(하강음)
- **Audio Mixer**: Master > BGM(-10dB) + SFX(0dB) 분리
