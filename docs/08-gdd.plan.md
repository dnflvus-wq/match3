# Phase 8: GDD (Game Design Document)

## 목표
게임의 전체 방향, 규칙, 콘텐츠, 수익화를 한 문서에 정리

## GDD 포함 항목

### 1. 게임 개요
- 장르: Match3 퍼즐
- 플랫폼: Android (향후 iOS)
- 타겟 유저: 캐주얼 모바일 게이머
- 핵심 재미: 연쇄 반응의 쾌감, 전략적 특수 타일 활용

### 2. 코어 게임플레이
- 기본 규칙 (3매치, 스와이프)
- 특수 타일 종류 및 효과
- 장애물 종류 (얼음, 사슬, 포털 등)
- 승리/패배 조건

### 3. 레벨 구조
- 총 레벨 수 목표
- 게임 모드별 규칙 (이동수/시간/장애물)
- 난이도 곡선

### 4. 메타게임
- 장기 목표 (스토리/꾸미기/수집)
- 월드맵 구조

### 5. 수익화
- IAP 구성
- 광고 배치
- 시즌패스

### 6. UI/UX 흐름
- 화면 전환 다이어그램
- 각 화면별 요소

### 7. 아트 스타일
- 테마 (판타지)
- 타일 디자인 방향
- 배경/UI 스타일 가이드

### 8. 사운드
- BGM 방향
- SFX 종류
