# Phase 13: 장애물 시스템

## 목표
다양한 장애물로 레벨 변화와 전략적 깊이 추가

## 장애물 종류

| 장애물 | 동작 | 파괴 방법 |
|--------|------|---------|
| 얼음 (1~3단계) | 타일 위에 레이어, 이동 불가 | 인접 매치 시 1단계씩 해빙 |
| 사슬 | 타일 고정, 스왑 불가 | 인접 매치 시 해제 |
| 금속 | 빈칸 차지, 파괴 불가 | 보드 형태 변형용 |
| 젤리 | 타일 아래 레이어 | 해당 타일 매치 시 제거 (레벨 목표) |
| 포털 | 입구→출구 타일 이동 | 파괴 불가, 경로 변경 |
| 타이머 폭탄 | N턴 후 게임오버 | 인접 매치로 제거 |

## 추가 기믹 (NotebookLM 확인)
- **컨베이어 벨트**: 타일을 한 방향으로 이동
- **생성기(대포/상자)**: 특정 타일을 주기적으로 뱉어냄
- **잠금 상자**: N번 매치하면 열리며 보상 아이템 드롭

## 비정형 보드
- 직사각형이 아닌 보드 형태 (구멍, 비대칭, L자 등)
- LevelConfig SO에 셀별 활성/비활성 플래그
- 금속 블록으로 구현하거나, 셀 자체를 비활성화

## 구현 방식
- **TileData에 obstacle 필드 추가** (ObstacleType enum + 단계 int)
- **장애물별 ScriptableObject**: 스프라이트, 파괴 조건, 점수
- **MatchFinder 확장**: 장애물 인접 매치 시 단계 감소 처리
- **Prefab**: 장애물 오버레이 스프라이트 (타일 위에 렌더링)

## 검증 항목
- [ ] 얼음 3단계 파괴 동작
- [ ] 사슬 해제 후 스왑 가능
- [ ] 금속 주변 낙하 우회
- [ ] 레벨 목표로 장애물 제거 카운트
