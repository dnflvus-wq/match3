# Phase 15: 출시 준비

## 목표
Google Play (향후 App Store) 출시에 필요한 모든 요소 준비

## 15-1. 다국어 지원 (Localization)
- Unity Localization 패키지 활용
- 지원 언어: 한국어, 영어 (초기)
- UI 텍스트, 튜토리얼 텍스트, 스토어 설명
- 폰트: 다국어 지원 폰트 (Noto Sans 등)

## 15-2. 접근성 (Accessibility)
- 색맹 모드 (타일에 형태/패턴 추가)
- 터치 영역 확대 옵션
- 텍스트 크기 조절
- 진동 피드백 on/off

## 15-3. 앱스토어 등록 (ASO)
- 앱 아이콘 (512×512, 1024×1024)
- 스토어 스크린샷 5~8장 (각 화면 하이라이트)
- 피처 그래픽 (1024×500)
- 프로모션 영상 (30초)
- 앱 설명 (한국어/영어)
- 키워드 최적화
- 연령 등급 심사

## 15-4. 약관/법률
- 개인정보처리방침
- 이용약관
- 결제 관련 고지 (미성년자 결제 제한)
- GDPR/CCPA 대응 (글로벌 출시 시)

## 15-5. 앱 업데이트 체계
- 강제 업데이트 (최소 버전 체크)
- 선택 업데이트 (권장 알림)
- 서버에서 최소 버전 관리

- [ ] 색맹 모드 타일 구분 가능
- [ ] 스토어 에셋 전부 준비
- [ ] 개인정보처리방침 URL 등록
- [ ] 강제 업데이트 작동
